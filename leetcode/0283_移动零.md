### 题目
给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。
### 示例
```js
输入: [0,1,0,3,12]
输出: [1,3,12,0,0]
```
### 说明：
1. 必须在原数组上操作，不能拷贝额外的数组。
2. 尽量减少操作次数。
```js
// 思路：移动非零元素，j记录下一个非零元素的位置，并把j位置非零元素移动至当前的i位置。
var moveZeroes = function(nums) {
    let j = 0;
    for (let i = 0, n = nums.length; i < n; i++) {
        if (nums[i] !== 0) {
            nums[j] = nums[i]
            if(i !== j) {
                nums[i] = 0
            }
            j++
        }
    }
};
```
